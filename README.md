# Expense Tracker App 💰

## Overview
A modern and interactive Expense Tracker web application that helps users manage income and expenses with real-time balance updates and category-wise tracking.

## Features
- ➕ Add income and expense transactions  
- 🧮 Real-time balance calculation  
- 📊 Expense visualization using charts (Chart.js)  
- 🗂️ Category-based tracking system  
- 💾 Data stored in browser using LocalStorage  
- 🧹 Reset all transactions option  
- 🗑️ Delete individual transactions  

## Tech Stack
- HTML5  
- CSS3  
- JavaScript (ES6)  
- Chart.js  

## How to Run
1. Download or clone the repository  
2. Open `index.html` in any browser
